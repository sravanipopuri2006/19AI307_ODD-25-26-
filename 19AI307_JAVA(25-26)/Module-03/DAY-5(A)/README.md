# Ex.No:3(E) INNER CLASS

## QUESTION:
Write a Java program to create an inner class and access it from the outer class.


## AIM:
To write a Java program that demonstrates the use of inner classes, where the inner class prints a greeting message using a name provided by the user

## SOURCE CODE:
```
import java.util.*;

class outer
{
    class inner
    {
        void print(String name)
        {
            System.out.print("Hello, "+name+"! This message is from the Inner Class.");
        }
    }
}

public class Main
{
    public static void main(String[] args)
    {
        Scanner s = new Scanner(System.in);
        String name = s.nextLine();
        
        outer out = new outer();
        outer.inner in = out.new inner();
        in.print(name);
    }
}
```
